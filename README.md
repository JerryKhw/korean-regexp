# korean-regexp

> 한글 정규표현식 Go 패키지

## Description

https://github.com/bluewings/korean-regexp

해당 Github를 보고 TypeScript로 구현된 한글 정규표현식 패키지를 Go로 구현한 패키지입니다.

## Installation

```
$ go get github.com/JerryKhw/korean-regexp
```
